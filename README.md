# ECommerce

A backend E-Commerce API built with ASP.NET Core, EF Core, and PostgreSQL.  
Designed as a portfolio project to demonstrate clean layered architecture, DTO-based data flow, repository/service patterns, and real-world backend development practices.

Implemented:
- JWT Authentication
- User registration / login
- User management
- Layered architecture: API / BLL / DAL
- DTO-based request/response flow
- Repository and Service layers
- EF Core + PostgreSQL
- Global exception handling middleware
- Initial domain structure for products / cart / orders / etc.

Project status:
Core structure is implemented. Some features are still being polished and debugged.
